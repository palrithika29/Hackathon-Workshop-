# 💰 Personal Finance Tracker (Hackathon Workshop)

## 📌 Project Description
This project was created during a Hackathon Workshop.  
It analyzes personal financial data using Python and Excel to understand spending patterns and calculate savings.

The project reads financial transaction data from Excel and performs:
- Expense tracking
- Category-wise spending analysis
- Monthly spending summary
- Savings calculation
- Data visualization

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Excel
- Jupyter Notebook

---

## 📂 Project Files

Finance_Tracker_Workshop_Excel.xlsx  
Financial_planning(Hackathon).ipynb  
Monthly_summary(Hackathon).xlsx  
Hackathon_Website.txt  

---

## ⚙ How to Run the Project

1. Install required libraries

```
pip install -r requirements.txt
```

2. Open Jupyter Notebook

```
jupyter notebook
```

3. Run the notebook

```
Financial_planning(Hackathon).ipynb
```

## 📊 Project Features

- Track personal expenses
- Analyze spending categories
- Calculate monthly expenses
- Compare expenses with savings goal
- Visualize spending using charts
